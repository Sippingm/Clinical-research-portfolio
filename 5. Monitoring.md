# Monitoring Tools & Checklists  

This section contains standardized monitoring checklists for different visit types.  

## Files  
- `Monitoring_Checklist_StartUp.md`  
- `Monitoring_Checklist_Routine.md`  
- `Monitoring_Checklist_CloseOut.md`  
# Monitoring Checklist – Start-Up Visit  

- Verify the completeness of regulatory documents completeness  
- Confirm IRB approval  
- Review site staff training (GCP, protocol, safety)  
- Confirm equipment and drug supply readiness  
- Validate informed consent process setup  
# Monitoring Checklist – Routine Monitoring Visit  

- Source data verification (SDV)  
- Adverse event reporting compliance  
- Drug accountability records  
- Screening/enrollment status vs. targets  
- Protocol deviations log review  
# Monitoring Checklist – Close-Out Visit  

- Ensure all CRFs and queries resolved  
- Verify drug returns/reconciliation  
- Archive essential documents (TMF/ISF)  
- Confirm database lock status  
- Conduct PI sign-off meeting
